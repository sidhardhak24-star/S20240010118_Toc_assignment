Core Concepts
Decidable
A Turing Machine halts on every input with accept or reject. The language is recursive. Example: aⁿbⁿcⁿ is decidable.
Recognizable (RE)
A TM accepts strings in the language, but for strings outside it may loop forever. Every decidable language is also recognizable.
Undecidable
No TM can correctly solve the problem for all inputs. Even with unlimited time, no algorithm exists.
Halting Problem
Given any TM M and input w, does M halt on w? Proven undecidable by Turing (1936) via diagonalization — the most famous undecidable problem.
